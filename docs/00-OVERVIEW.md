# 00. 개요 (Overview)

**blumnAI-qa-bot** 은 회사 안 각 서비스 팀이 이 봇을 가져다가 본인 정책문서와 연결해서, AI 와 대화하며 변경 요청을 자동 합의문화할 수 있게 해주는 재사용 가능한 QA 봇 툴킷입니다.

## 무엇을 해주나

QA·운영팀이 매일 겪는 문제 — *"이 화면이 정책상 어떻게 동작해야 하는지 누구한테 물어야 하지?"* — 를 해결합니다.

1. **정책 질문 → AI 가 즉시 답변**
   협업자(QA·운영)는 정책문서·화면설계서를 보며 채팅창에 질문. 봇이 그 문서를 근거로 답해줌. (예: *"대시보드 알림 배너 닫기 동작 어떻게 돼?"* → *"주황 배너는 닫으면 24시간 미표시. 빨강 배너는 닫기 불가. §2-2 참고"*)

2. **답변이 애매하거나 정책 자체를 바꿔야 하면 → [📤 기획자에게 전달]**
   기획자가 검토 후 ✅ 적용 / 🚫 보류 처리. 모든 합의는 `qa/decisions/` 에 markdown 으로 자동 저장.

3. **답변 톤이 불편하면 → [📝 개선 요청]**
   *"5줄 안으로 요약해줘", "기술용어 빼고"* 같은 요청. `qa/feedback/` 에 누적되어 이후 답변에 자동 반영.

## 아키텍처 한 장

```
┌─────────────────────────────────────────────────────────────┐
│ 협업자 PC                                                    │
│ ┌────────────────┐                                          │
│ │ 브라우저       │  qa-collab.html  (GitHub Pages 호스팅)     │
│ │                │  qa-planner.html (기획자만)               │
│ └───────┬────────┘                                          │
└─────────┼───────────────────────────────────────────────────┘
          │
          ▼  (fetch)
┌─────────────────────────────────────────────────────────────┐
│ Cloudflare Worker  (팀당 1개, 무료 tier)                     │
│   GET  /list-projects · /list-docs · /doc                   │
│   POST /qa · /forward · /feedback                           │
└─────────┬───────────────────────────────────────────────────┘
          │
          ├─ [기본] A 모드 ─→  Anthropic API (sk-ant-... 유료, 월 ~$30)
          │                    + GitHub API (정책 read, decisions/feedback write)
          │                    + (옵션) GitHub Code Search — 코드 레포 검증
          │
          └─ [부록] C 모드 ─→  본인 PC tunnel
                                └─ Claude Code CLI (Max 구독, 무료)
```

## 두 가지 운영 모드 — A 모드가 기본

| | **A 모드 (기본)** | C 모드 (부록) |
|---|---|---|
| 방식 | Anthropic API key 로 Worker 가 직접 호출 | 본인 PC 의 Claude Max 구독을 tunnel 로 우회 |
| 비용 | API 토큰당 과금. 월 100질문 ≈ $15-30 | 0원 (Max 구독 한도 안에서) |
| PC 의존성 | 없음 (Worker 가 다 처리) | PC + `start.bat` 콘솔창 항상 켜져 있어야 |
| 응답 속도 | 3-5초 | 8-15초 (claude CLI 오버헤드) |
| 셋업 스텝 | 시크릿 2개 등록 + 배포 | + Claude CLI 설치 + cloudflared + tunnel 유지 |
| 미래 위험 | 없음 | Anthropic 정책 변경 시 막힐 수 있음 (회색지대) |

→ **A 모드로 시작.** 회사 Anthropic 결제만 받으면 셋업이 훨씬 단순하고, PC 상시 가동·터널 유지 부담이 없습니다. 결제 확보가 어렵거나 개인 Max 구독으로 우선 검증해보고 싶은 특수 상황에서만 C 모드를 고려하세요 (자세한 설명은 [03-CONNECT-BOT §부록](03-CONNECT-BOT.md#부록--c-모드-pc-max-우회-특수-상황용)).

## 사용자 레포 = 정책 SSoT

이 봇은 재사용 가능한 코어 레포이고, 각 서비스 팀이 본인 정책 레포에 코어 사본을 두고 컨피그 파일 하나로 연결해서 씁니다. 즉:

- 코어 봇 레포(`blumnAI-qa-bot`) — Worker 코드, HTML, 가이드 (모든 팀이 공유)
- 광고팀 정책 레포(`ad-team-policies` — 광고팀이 만듦) — 광고팀 정책 md + `blumnAI-qa-bot.config.yml` (봇 연결 컨피그) + 코어 사본(`.blumnAI-qa-bot/`) + 자동 생성되는 `qa/decisions/`

광고팀은 봇 레포에서 코어를 가져다 자기 정책 레포에 두고, 컨피그 1개만 채우면 됩니다.

## 무엇을 얻나

- **즉답** — 정책 질문 → 챗박스 한 번 → 7초 답변. Slack 에서 누구 찾아갈 필요 없음
- **자동 합의 기록** — 변경 요청을 입으로만 약속하지 않음. `qa/decisions/` 에 markdown 으로 박힘. 누가 언제 무엇을 왜 바꿔달라고 했고, 기획자가 어떻게 처리했는지 git history 로 추적 가능
- **답변 품질 자가 학습** — 한 번 받은 개선 요청은 이후 모든 답변에 자동 반영. 같은 실수 반복 안 함

## 🤝 설치·운영은 AI 에이전트와 함께

이 봇의 설치 가이드는 **Claude Code 같은 AI 에이전트와 함께 진행하는 것을 전제**로 설계되어 있습니다. 다른 서비스팀의 담당자가 대부분 비개발자여도, 옆에 Claude Code 창을 열고 다음 두 종류의 작업을 구분해서 진행하면 됩니다:

- **🤖 AI 에게 위임** — 폴더 만들기, 파일 복사, 컨피그 편집, git·wrangler 명령어 실행, 트러블슈팅 등. AI 가 알아서 처리하고 결과를 보고합니다
- **👤 본인이 직접** — 웹 브라우저 접속 (GitHub 레포 생성, Cloudflare 가입, Pages 활성화), 시크릿 값 발급 (Anthropic API key, GitHub PAT), wrangler 프롬프트에 시크릿 값 붙여넣기, 최종 브라우저 동작 확인

> 🔒 **시크릿 값 (API key, PAT) 은 절대 AI 채팅창에 붙여넣지 마세요.** 대화 로그에 남습니다. wrangler 프롬프트 (터미널에서 "Enter a secret value:") 에만 직접 입력.

가이드의 각 스텝 앞에 🤖 / 👤 아이콘이 붙어 있으니, 자기 담당 작업만 하면 됩니다. 처음 시작은 [01-INSTALL.md](01-INSTALL.md#-quick-start--마스터-프롬프트-하나로-시작) 의 Quick Start 마스터 프롬프트를 Claude Code 에 붙여넣으면 됩니다.

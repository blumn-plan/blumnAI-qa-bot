# 대시보드 v0.1.0

샘플 정책문서 — 봇이 이 문서를 근거로 답변할 수 있는지 동작 테스트용.

## §1. 화면 구성

대시보드는 다음 4개 영역으로 구성된다:

1. 상단 알림 배너
2. 핵심 지표 카드 (좌)
3. 시계열 차트 (우)
4. 최근 활동 리스트 (하단)

## §2. 알림 배너 정책

### §2-1. 표시 조건

- 잔액이 발송 가능 한도 미만이면 주황 배너 표시
- 외부 연동이 끊겼으면 빨강 배너 표시 (닫기 불가)

### §2-2. 닫기 동작

- 주황 배너 — 사용자가 닫으면 24시간 동안 미표시
- 빨강 배너 — 닫기 버튼 없음 (재연결 전까지 계속 노출)

## §3. 핵심 지표 카드

오늘 발송 건수 / 오늘 사용 포인트 / 잔여 포인트 / 응답률 — 4개 카드 표시.
